# sales-dashboard
Interactive E-commerce Sales Dashboard built using Streamlit, Python, and data visualization libraries to analyze sales trends and customer behavior.
# E-commerce Sales Dashboard
#🔗 Live Demo

# Project Overview
This project is an interactive sales dashboard built using Streamlit. It helps analyze e-commerce sales data and provides insights into sales trends, customer behavior, and product performance.

# Objectives
- Analyze sales performance
- Understand customer behavior
- Identify top-selling products
- Visualize data using interactive charts

# Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit

#Features
- Total Sales, Orders, and Average Rating (KPIs)
- Category-wise Sales Analysis
- Monthly Sales Trend
- Top 10 Products
- Payment Method Distribution
- Customer Rating Analysis
- Interactive Filters

# Dataset
The dataset includes:
- Order Date
- Category
- Product Name
- Payment Method
- Total Sales (INR)
- Customer Rating
